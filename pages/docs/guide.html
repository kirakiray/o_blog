介绍

准备

首先，你得先学会最基础的 html css js 知识，例如新建一个空白页。

其次，建议学习一下 ES6的 Promise API 和 ES7的 async function。

然后，将ofa.js添加到你的页面中；

接下来就可以开始正式的教程了。

为了提高教学质量，这里将ofajs的教学分5个阶段，每个阶段很轻松完成学习，学完都能有所收获。

教程目录

stage 0: ofajs封装的组件的使用

讲解如何使用ofajs封装的组件；只需要学习ofajs提供的 load 方法即可；

stage 0 学习完成后，你也能 五分钟开发一个markdown编辑器。

立刻开始

stage 1: 使用ofa api

ofajs提供的api跟jQuery很相似，是jQuery的精神续作；

如果你喜欢jQuery，一定也会喜欢ofajs；

stage 1 完成后，能够快速开发页面的小功能。

stage 2: 使用ofajs开发组件

讲解如何开发ofajs组件；(就是 stage 0 中的组件)

stage 2完成后，能开发出让别人快速使用的组件；

stage 3: 使用ofajs开发页面

学习stage3之前，必须先学习过stage2的内容；

stage 3 完成后，能够快速开发功能页面；

stage 4: 使用ofajs开发app

学习stage4之前，必须先学习stage3的内容；

stage4完成后，你就能开发全平台通用的高体验应用程序（PWA or Webapp）。
